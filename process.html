Collect Data from Kaggle And ESRI

We have collected the USA accidents dataset between 2016 and 2021 from Kaggle .
Also, We collected the road network dataset from ESRI .

Prepare Network layer

1- Spatial join between the network layer and accidents data
2- generate tesselation
3- Hot spot and outlier analysis
4- Emerging hotspot and outlier analysis

Prepare the Accidents dataset and create a model

1- Use colab
2- Check Nulls and Duplicates vales
3- Drop unnecessary columns
4- Calculate columns (Eculidiun Lenght - Sinuisty - Road category )
5- Check the distribution of attributes
6- Deal with nan values
7- Create a target label
8- Data splitting and balancing
9- Choose and train model

Create Application Server and Deploy Model

1- Installing flask and preparing the virtual environment ,
2- load model using Pickle
3- Create an application server and handle routing
4- Receive the request and convert it to a 2D array
5- We put our code on Docker image then upload to Docker hub
6- From Docker hub we pull this image on EC2
7- We deploy this image as a Docker container on AWS

Create Map View

1- Use ArcGIS API for JS v4
2- Publishing network dataset on Esri server
3- Making Esri request for the user's current extent
4- Making Restfull API requests from OpenWeathermap.com with road long and latitude
5- taking user time input currently or 24 hours later
6- Create JS Object with required params to send to the model
7- Receive model result and visualize it
8- Creating polyline barriers layer for further analysis
